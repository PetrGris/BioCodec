Техническое задание (ТЗ)
Проект: BIOHACKER — портал для биохакеров

1. Общая концепция
BIOHACKER — это современный веб-портал для людей, интересующихся биохакингом, здоровьем, саморазвитием и персонализированной медициной. Портал объединяет инструменты для мониторинга здоровья, аналитики, библиотеку знаний, ИИ-консультантов и новостную ленту.

2. Стек технологий
- Frontend: React (Vite), Material UI, Chart.js, React Router, Axios
- Backend: FastAPI, SQLAlchemy, Alembic, Pydantic, SQLite/PostgreSQL, OAuth2, интеграция с YandexGPT
- CMS: Strapi (опционально)
- Документация: Markdown, OpenAPI
- Хранение файлов: S3-совместимое хранилище (опционально)

3. Структура проекта
BIOHACKER/
├── frontend/      # Веб-интерфейс
├── backend/       # API и бизнес-логика
├── cms/           # Контент-менеджмент (опционально)
├── docs/          # Документация, схемы, инструкции

4. Основные разделы и страницы
- Новостная лента: карточки, фильтры, поиск, категории, дата, иконки действий
- Личный кабинет (Dashboard):
  • Панель мониторинга: графики, сводки, сравнения
  • Корреляционный анализ: взаимосвязи, прогнозы
  • Отчёты и экспорт: генерация, экспорт, поделиться
  • Ручной ввод параметров: антропометрия, самочувствие, питание, добавки, активность, сон, стресс, когнитивные тесты
  • Демо-режим: просмотр без регистрации
- Библиотека: карточки в стиле Notion, фильтры, fallback-данные
- Домены биохакинга: отдельные страницы по темам (сон, питание, стресс, долголетие, когнитивные функции, физическая активность, генетика)
- ИИ-консультанты (нейроконсультанты):
  • Каждый консультант — отдельный пункт меню: коуч по биохакингу, ИИ-нутрициолог, консультант по спорту
  • Страница: крупный аватар, фон, карточка с фишками, быстрые советы, красивый чат, интеграция с YAGPT через индивидуальный промпт

5. Взаимодействие компонентов
- Frontend: единый дизайн, адаптивность, навигация через React Router, графики Chart.js, чат с ИИ через API
- Backend: FastAPI REST API, авторизация OAuth2, endpoint для чата с YAGPT (user prompt + system prompt)
- Интеграция с YAGPT: для каждого консультанта свой system prompt (consultantPrompts.js), user prompt — сообщение пользователя

6. Развёртывание
Backend:
  cd backend
  python -m venv .venv
  .venv\Scripts\activate
  pip install -r requirements.txt
  alembic upgrade head
  uvicorn app.main:app --reload
Frontend:
  cd frontend
  npm install
  npm run dev
CMS (опционально):
  cd cms
  npm install
  npm run develop

7. Переменные окружения
- Backend: .env (ключи БД, OAuth, YandexGPT)
- Frontend: .env (API_URL, ключи интеграций)

8. Примеры данных и демо-режим
- Для неавторизованных пользователей — демо-данные
- Для реальных данных — регистрация и анкета

9. Частые проблемы и FAQ
- Pydantic: предупреждение про orm_mode — заменить на from_attributes
- CORS/ECONNREFUSED: проверить запуск backend и frontend, адреса API
- 500/404: проверить миграции, структуру БД, логи backend
